# Example python and c++ code in linux mint

###  Commands in linux

#### File creation and updating `touch` and `nano`
* Create file: *touch*
examples

```
touch <name>.py 
touch <name>.cpp
```

* Create file and edit: *nano*

examples
```
nano <name>.py
nano <name>.cpp
```

#### Run or compile for Python and C++
* Python
```
python3 <name>.py
```

* C++
```
g++ <name>.cpp -o <name_executable>
./<name_executable>
```
